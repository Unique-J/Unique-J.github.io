发表于 2016-10-25   |   分类于 Markdown   |     |   次阅读

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。

标题

在 Markdown 中你可以在标题前加上 # ,一个 #代表一级标题,两个 #代表二级标题,以此类推。在 Markdown 中总共有六级标题。
